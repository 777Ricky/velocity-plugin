# Analyse (Velocity Plugin)

This is the official velocity adapter for Analyse, the Minecraft Server & Player Analytics platform that tracks community performance.

## Installation
1. Download this plugin (Latest from Releases).
2. Start the proxy.
3. Edit the `plugins/analyse/config.yml` file with your Redis credentials.
4. Restart your proxy.
5. Make sure your Spigot servers have the same Redis credentials.

You're ready to go!
